# App 首页与我的页面 - 原型使用说明

## 📱 原型概览

本原型展示了移动应用的两个核心 TAB 页面：**首页**和**我的**页面。采用微信小程序原生风格设计，支持游客模式和登录模式切换。

## 🎯 设计系统

- **UI 风格**：微信小程序原生风格
- **主色调**：Tech Blue (#3478F6)
- **字体**：PingFang SC, -apple-system
- **目标平台**：微信小程序（优先）、H5（优先）

## ✨ 功能演示

### 首页功能

#### 1. 轮播图
- **自动播放**：每 4 秒自动切换
- **手动滑动**：支持左右滑动切换（移动端）
- **指示器**：底部圆点显示当前位置
- **点击交互**：点击轮播图显示 Toast 提示

#### 2. 内容列表
- **内容卡片**：展示封面图、标题、摘要、发布时间
- **标签系统**：推荐、热门等标签
- **点击交互**：点击卡片显示 Toast 提示
- **加载提示**：底部显示"下拉刷新，上拉加载更多"

#### 3. 底部导航
- **双 TAB 切换**：首页 / 我的
- **高亮状态**：当前页面图标和文字高亮显示

### 我的页面功能

#### 1. 用户信息区
- **头像展示**：圆形头像，点击可编辑
- **用户信息**：昵称、会员等级
- **数据统计**：积分、余额（可点击查看详情）
- **游客模式**：未登录时显示"登录/注册"按钮

#### 2. 功能菜单
- **个人信息**：进入个人信息编辑页
- **账号安全**：修改密码、绑定手机、注销账号
- **隐私设置**：隐私协议、用户协议、权限管理
- **语言切换**：支持中文/英文切换
- **清除缓存**：显示缓存大小，点击清除
- **关于我们**：应用介绍和联系方式
- **版本更新**：显示当前版本，检查更新

#### 3. 退出登录
- **二次确认**：点击后弹出确认对话框
- **状态切换**：退出后切换到游客模式

## 🎯 用户流程

### 游客浏览流程
```
打开应用 → 查看首页轮播图和内容 → 点击内容卡片 →
```

```
打开应用 → 查看首页内容 → 切换到"我的"页面 →
```


## 📝 设计决策说明

### 1. 为什么选择微信小程序风格？
- PRD 明确要求微信小程序原生风格
- 目标平台优先级：微信小程序 > H5
- 用户熟悉度高，学习成本低

### 2. 为什么使用渐变色轮播图？
- 提升视觉吸引力
- 区分不同内容类型
- 符合现代设计趋势

### 3. 为什么支持游客模式？
- PRD 要求支持游客浏览
- 降低使用门槛
- 提高用户注册转化率

### 4. 为什么使用 Toast 和模态框？
- 符合微信小程序交互规范
- 提供即时反馈
- 重要操作需要二次确认
