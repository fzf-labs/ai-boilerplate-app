# PRD：App 首页与我的页面

## 执行摘要

本文档定义了移动应用的两个核心 TAB 页面：**首页**和**我的**页面的产品需求。该应用是一款工具/服务类移动应用，旨在通过内容展示和用户服务实现品牌宣传、工具使用和用户增长三大核心目标。

**核心价值主张**：
- 为用户提供有价值的内容资讯和推荐
- 提供便捷的个人信息管理和账号设置
- 支持游客浏览和会员登录双模式，降低使用门槛

**目标平台**：微信小程序（优先）、H5（优先）

**质量评分**：93/100

---

## 1. 问题陈述

### 1.1 业务背景
当前移动应用市场竞争激烈，用户对应用的首次体验和个人信息管理有较高要求。我们需要构建一个既能吸引新用户（通过内容展示），又能留住老用户（通过个性化服务）的移动应用。

### 1.2 用户痛点
1. **新用户**：不了解应用价值，需要快速了解核心功能和内容
2. **游客用户**：希望先体验内容，再决定是否注册
3. **注册用户**：需要便捷管理个人信息、查看会员权益、调整应用设置
4. **所有用户**：希望界面简洁、操作流畅、符合微信小程序使用习惯

### 1.3 解决方案
通过设计清晰的首页内容展示和完善的个人中心功能，提供：
- **首页**：轮播图 + 内容列表的混合展示，快速传递核心信息
- **我的页面**：用户信息 + 设置功能的分层结构，满足不同层次需求
- **游客模式**：允许未登录用户浏览内容，降低使用门槛

## 4. 用户故事

### 4.1 首页相关

**US-001：作为新用户，我希望在首页看到轮播图，以便快速了解应用的核心功能和最新活动**
- **验收标准**：
  - 轮播图自动播放，间隔 3-5 秒
  - 支持手动滑动切换
  - 点击轮播图跳转到对应详情页
  - 显示轮播图指示器（圆点）

**US-002：作为用户，我希望在首页看到内容列表，以便浏览感兴趣的资讯和推荐**
- **验收标准**：
  - 内容列表显示标题、摘要、封面图、发布时间
  - 支持下拉刷新获取最新内容
  - 支持上拉加载更多内容
  - 点击内容卡片跳转到详情页

**US-003：作为游客用户，我希望无需登录即可浏览首页内容，以便先体验应用**
- **验收标准**：
  - 游客可以查看首页所有内容
  - 部分功能（如收藏、评论）提示需要登录
  - 提供明显的登录/注册入口

### 4.2 我的页面相关

**US-004：作为注册用户，我希望在"我的"页面看到个人信息，以便了解和编辑我的资料**
- **验收标准**：
  - 显示头像、昵称、性别、生日
  - 显示手机号、邮箱、地址
  - 显示会员等级、积分、余额
  - 点击可进入编辑页面

**US-005：作为用户，我希望在"我的"页面管理账号安全，以便保护我的账号**
- **验收标准**：
  - 提供修改密码功能
  - 提供绑定/解绑手机号功能
  - 提供注销账号功能（需二次确认）

**US-006：作为用户，我希望在"我的"页面调整应用设置，以便个性化使用体验**
- **验收标准**：
  - 提供语言切换功能
  - 提供清除缓存功能
  - 提供查看隐私协议和用户协议功能
  - 提供关于我们和版本更新信息

**US-007：作为游客用户，我希望在"我的"页面看到登录提示，以便快速注册或登录**
- **验收标准**：
  - 显示"未登录"状态
  - 提供明显的登录/注册按钮
  - 部分功能显示为灰色不可用状态

## 5. 功能需求

### 5.1 首页功能

#### 5.1.1 轮播图模块
**功能描述**：展示重要公告、活动推广、品牌宣传等内容

**交互规则**：
- 自动播放，间隔 3-5 秒
- 支持手动左右滑动切换
- 显示指示器（圆点），当前页高亮
- 点击跳转到对应详情页或外部链接

**数据来源**：后端接口返回轮播图列表（图片 URL、跳转链接、排序）

**边界情况**：
- 无轮播图数据时，隐藏该模块
- 只有 1 张轮播图时，不显示指示器，不自动播放
- 图片加载失败时，显示占位图

#### 5.1.2 内容列表模块
**功能描述**：展示资讯、推荐内容等信息流

**内容卡片结构**：
- 封面图（左侧或顶部）
- 标题（1-2 行，超出省略）
- 摘要（2-3 行，超出省略）
- 发布时间（相对时间，如"2小时前"）
- 标签（可选，如"推荐"、"热门"）

**交互规则**：
- 下拉刷新：获取最新内容
- 上拉加载：加载更多内容（分页）
- 点击卡片：跳转到内容详情页

**数据来源**：后端接口返回内容列表（分页）

**边界情况**：
- 无内容时，显示空状态提示
- 加载失败时，显示错误提示和重试按钮
- 已加载全部内容时，显示"没有更多了"

#### 5.1.3 底部导航栏
**功能描述**：提供首页和我的页面的快速切换

**导航项**：
- 首页（图标 + 文字）
- 我的（图标 + 文字）

**交互规则**：
- 当前页面高亮显示
- 点击切换到对应页面

### 5.2 我的页面功能

#### 5.2.1 用户信息区
**功能描述**：展示用户基本信息和会员状态

**未登录状态**：
- 显示默认头像
- 显示"未登录"文字
- 显示"登录/注册"按钮

**已登录状态**：
- 显示用户头像（圆形）
- 显示昵称
- 显示会员等级标识（如 VIP、普通会员）
- 显示积分和余额（可点击查看详情）

**交互规则**：
- 点击头像或昵称：进入个人信息编辑页
- 点击积分/余额：进入积分/余额详情页

#### 5.2.2 个人信息编辑
**功能描述**：编辑用户基本信息和联系方式

**可编辑字段**：
- 头像（点击上传）
- 昵称（文本输入，限制 2-20 字符）
- 性别（单选：男/女/保密）
- 生日（日期选择器）
- 手机号（文本输入，验证格式）
- 邮箱（文本输入，验证格式）
- 地址（文本输入或地址选择器）

**交互规则**：
- 点击字段进入编辑模式
- 修改后点击保存按钮提交
- 验证失败时显示错误提示

**边界情况**：
- 昵称重复时提示
- 手机号/邮箱格式错误时提示
- 保存失败时显示错误信息

#### 5.2.3 账号安全设置
**功能描述**：管理账号安全相关功能

**功能列表**：
1. **修改密码**
   - 输入旧密码
   - 输入新密码（6-20 位，包含字母和数字）
   - 确认新密码
   - 验证通过后修改

2. **绑定手机号**
   - 输入手机号
   - 发送验证码
   - 输入验证码验证
   - 绑定成功

3. **注销账号**
   - 显示注销说明（数据将被删除）
   - 二次确认弹窗
   - 输入密码验证
   - 注销成功后退出登录

#### 5.2.4 隐私设置
**功能描述**：查看协议和管理权限

**功能列表**：
- 隐私协议（跳转到协议页面）
- 用户协议（跳转到协议页面）
- 权限管理（查看和管理应用权限）

#### 5.2.5 通用设置
**功能描述**：应用的通用配置项

**功能列表**：
1. **语言切换**
   - 支持中文/英文切换
   - 切换后立即生效

2. **清除缓存**
   - 显示当前缓存大小
   - 点击清除后显示成功提示

3. **关于我们**
   - 显示应用介绍
   - 显示公司信息
   - 显示联系方式

4. **版本更新**
   - 显示当前版本号
   - 检查是否有新版本
   - 有新版本时提示更新

#### 5.2.6 退出登录
**功能描述**：退出当前账号

**交互规则**：
- 点击"退出登录"按钮
